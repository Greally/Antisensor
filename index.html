📋 About This Tool

This tool assesses genetic variants for suitability for three ASO therapy approaches:

  • RNA Knockdown: Selectively reduce mutant allele expression (for gain-of-function variants)
  • Splice Correction: Restore normal splicing patterns (for splice site variants)
  • Exon Skipping: Skip problematic exons to restore reading frame (for frameshifts)
